# StudyOS

🚀 AI-powered JEE study command center built with pure HTML/CSS/JS.

Pomodoro timer · Subject tracking · AI Tutor (NVIDIA NIM) · Weekly planner · Galaxy visualization

## Files

- `index.html` — Main app (Stitch AI aesthetic, glassmorphism, spring animations)
- `galaxy.html` — 3D galaxy visualization with raycasting & CatmullRom paths

## Live Preview

Open `index.html` directly in any modern browser.
